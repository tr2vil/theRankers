## Sprint 8: 핵심 차별화 기능 (즉시 도입 가능)

난이도 하 — 기존 데이터/모델로 바로 구현 가능한 기능들.

| 작업 | 설명 | 참고 |
|------|------|------|
| Best Analyst Filter | 상위 랭킹 애널리스트만의 컨센서스 별도 표시 (종목 상세) | TipRanks Premium 핵심 기능 |
| 증권사(Firm) 랭킹 | 증권사별 종합 성과 랭킹 페이지 | Analyst.firm 집계 |
| 섹터별 필터링 강화 | 랭킹/애널리스트/종목 페이지에 GICS 섹터 필터 추가 | 이미 GICS 11개 매핑 완료 |
| 기술적 분석 지표 | SMA 20일/200일, 12개월 모멘텀 (종목 상세) | pykrx 주가 데이터로 계산 |
| theRankers Top 20 Index | 상위 평가 종목 자체 인덱스 (메인 대시보드) | 브랜딩 + 마케팅 효과 |

## Sprint 9: Smart Score 한국형 + 비교 도구

난이도 중 — 신규 데이터 수집 또는 중간 규모 UI 작업 필요.

| 작업 | 설명 | 참고 |
|------|------|------|
| Smart Score 한국형 | 컨센서스 + 기관/외인 수급 + 기술적 지표 + 펀더멘탈 결합 1-10점 | TipRanks 킬러 피처의 한국 변형 |
| 종목 비교 도구 | 최대 5개 종목 병렬 비교 (컨센서스, 목표가, 주가 차트) | 사용자 경험 크게 향상 |
| 포트폴리오 트래커 | 보유 종목 등록 + 실시간 애널리스트 의견 모니터링 | User 모델 확장 필요 |
| 의견 변경 알림 | 리포트 diff 감지 + 이메일 알림 | Celery 수집 파이프라인에 diff 로직 추가 |
| AI 종목 리포트 | LLM으로 종목 장단점 요약 자동 생성 | Claude API 활용, 비용 고려 |
