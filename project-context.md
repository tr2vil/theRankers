# theRankers - 프로젝트 컨텍스트

## 비전
국내 증권 시장의 "TipRanks"를 만든다. 애널리스트 리포트의 투자의견과 목표가를 체계적으로 추적하여, 누가 실제로 잘 맞추는지를 데이터로 증명하는 서비스.

## 문제 정의
1. **매수 편향**: 국내 애널리스트 리포트의 90%+ 가 "매수" 의견 → 변별력 없음
2. **검증 부재**: 목표가를 제시하지만 달성 여부를 추적하는 시스템이 없음
3. **정보 비대칭**: 개인 투자자는 애널리스트의 과거 성적을 알 수 없음

## 핵심 기능 (TipRanks 벤치마킹)

### MVP (Phase 1)
- 네이버 금융 컨센서스 데이터 수집
- 애널리스트별 리포트 이력 저장
- 주가 추적 (리포트 발행 후 1M/3M/6M/12M)
- 기본 스코어링 (목표가 달성률 + 수익률)
- 애널리스트 랭킹 테이블

### 확장 (Phase 2)
- 복수 데이터 소스 (한경, 증권사 직접)
- 섹터별/증권사별 랭킹
- 애널리스트 프로필 페이지
- 종목별 애널리스트 의견 집계
- 검색 및 필터링

### 고도화 (Phase 3)
- 실시간 알림 (관심 애널리스트 신규 리포트)
- 포트폴리오 시뮬레이션 (상위 랭커 따라가기)
- API 공개
- 모바일 앱

## TipRanks 참조 기능

### 애널리스트 랭킹
- Star Rating (1~5)
- Success Rate (수익 낸 추천 비율)
- Average Return (평균 수익률)
- 기간별 필터 (1Y, 2Y, All)
- 섹터별 필터

### 종목 페이지
- 애널리스트 컨센서스 (매수/중립/매도 분포)
- 평균 목표가 vs 현재가
- 가장 정확한 애널리스트의 의견 하이라이트
- 목표가 히스토리 차트

### 애널리스트 프로필
- 전체 리포트 이력
- 종목별 성과
- 랭킹 변화 추이
- 소속 증권사 정보

## 차별화 전략
1. **한국 시장 특화**: KOSPI/KOSDAQ 이원 시장, 증권사 리서치센터 중심 구조
2. **매수 편향 보정**: 의견 분포의 극단적 편향을 보정한 실질 성과 평가
3. **섹터 난이도 보정**: 변동성 높은 섹터의 예측이 더 어려움을 반영
4. **투명한 방법론**: 스코어링 로직을 공개하여 신뢰 확보
